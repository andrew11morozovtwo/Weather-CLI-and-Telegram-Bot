# 🌦️ Weather Telegram Bot & CLI Tool

Универсальный погодный помощник, включающий консольную утилиту и продвинутого Telegram-бота с рассылкой по местному времени.

## 🌟 Основные возможности

### 🤖 Telegram Бот
*   **Рассылка по местному времени**: Бот автоматически определяет часовой пояс каждого города и отправляет прогноз ровно в 08:00 по местному времени.
*   **Интерактивный прогноз**: Кнопка под каждым сообщением для получения почасового прогноза на ближайшие 12 часов (с использованием линейной интерполяции данных).
*   **Админ-панель**: Управление списком городов (добавление/удаление) прямо через интерфейс Telegram.
*   **Безопасность**: Доступ к настройкам только для администратора (по Telegram ID).

### 💻 Консольная утилита (`weather.py`)
*   Интерактивный поиск погоды в терминале.
*   Подтверждение найденного города перед выводом данных.
*   Прогноз на 24 часа.

## 🚀 Быстрый старт

### 1. Требования
*   Python 3.8+
*   API ключ [OpenWeatherMap](https://openweathermap.org/api)

### 2. Установка
```bash
# Клонируйте репозиторий
git clone https://github.com/youruser/weather-bot.git
cd weather-bot

# Установите зависимости
pip install -r requirements.txt
```

### 3. Настройка
Создайте файл `.env` в корневой директории (используйте `test_env.example` как образец):
```env
OPENWEATHER_API_KEY=ваш_ключ_openweathermap
TELEGRAM_BOT_TOKEN=ваш_токен_от_botfather
ADMIN_ID=ваш_telegram_id
CHAT_ID=id_чата_для_рассылки
```

### 4. Запуск

**Запуск Telegram-бота:**
```bash
python bot.py
```

**Запуск консольной версии:**
```bash
python weather.py
```

## 🛠️ Технологический стек
*   **aiogram 3.x**: Асинхронная библиотека для Telegram API.
*   **requests**: Работа с HTTP-запросами к OpenWeatherMap.
*   **timezonefinder & pytz**: Точное определение часовых поясов по координатам.
*   **python-dotenv**: Управление конфигурацией через переменные окружения.
