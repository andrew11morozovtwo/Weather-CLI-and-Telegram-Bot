# 🌦️ Weather Telegram Bot

Продвинутый Telegram-бот для автоматической рассылки погоды по местному времени и интерактивного прогноза.

## 🌟 Основные возможности

*   **Рассылка по местному времени**: Бот автоматически определяет часовой пояс каждого города и отправляет прогноз ровно в 08:00 по местному времени.
*   **Интерактивный прогноз**: Кнопка под каждым сообщением для получения почасового прогноза на ближайшие 12 часов (с использованием линейной интерполяции данных). Прогноз приходит в личные сообщения пользователю.
*   **Автоматическое удаление сообщений**: Все сообщения бота автоматически удаляются через заданное время (по умолчанию 12 часов). Время настраивается через админ-панель.
*   **Админ-панель**: Управление списком городов (добавление/удаление) и настройка времени удаления сообщений прямо через интерфейс Telegram.
*   **Безопасность**: Доступ к настройкам только для администратора (по Telegram ID).

## 🚀 Быстрый старт

### 1. Требования
*   Python 3.8+
*   API ключ [OpenWeatherMap](https://openweathermap.org/api)

### 2. Установка
```bash
# Клонируйте репозиторий
git clone https://github.com/youruser/weather-bot.git
cd weather-bot

# Установите зависимости
pip install -r requirements.txt
```

### 3. Настройка
Создайте файл `.env` в корневой директории (используйте `test_env.example` как образец):
```env
OPENWEATHER_API_KEY=ваш_ключ_openweathermap
TELEGRAM_BOT_TOKEN=ваш_токен_от_botfather
ADMIN_ID=ваш_telegram_id
CHAT_ID=id_чата_для_рассылки
```

### 4. Запуск

**Запуск Telegram-бота:**
```bash
python bot.py
```

**Запуск оптимизированной версии для сервера:**
```bash
python bot1.py
```

*   **aiogram 3.x**: Асинхронная библиотека для Telegram API.

## ⚙️ Дополнительные функции

### Автоматическое удаление сообщений
Бот автоматически удаляет все свои сообщения через заданное время после отправки. Это помогает поддерживать чистоту в чате.

**Настройка времени удаления:**
1. Откройте админ-панель командой `/admin`.
2. Нажмите кнопку **"⚙️ Время удаления"**.
3. Введите время в формате `часы:минуты` (например: `12:30` для 12 часов 30 минут).
4. Бот сохранит настройку и будет использовать её для всех новых сообщений.

**Значение по умолчанию:** 12 часов (720 минут).

### Отправка прогноза в личные сообщения
При нажатии на кнопку **"📊 Прогноз на 12 часов"** прогноз отправляется пользователю в личные сообщения, а не в общий чат. Это позволяет каждому пользователю получать персональный прогноз без засорения группового чата.

**Важно:** Если пользователь еще не начинал диалог с ботом в личке, прогноз будет отправлен в группу с предупреждением.

### Служебные скрипты

**Очистка базы данных:**
Для тестирования на чистой базе используйте скрипт:
```bash
python clear_db.py
```
Этот скрипт удаляет файл `bot_messages.db`, позволяя начать тестирование с нуля.
